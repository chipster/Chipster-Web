Contact information


If you have questions about using or installing Chipster, we encourage you to email our community mailing lists chipster-users and chipster-tech (for details see below). You don't need to be subscribed to send or view messages. For more confidential enquiries please contact the server maintainers or email the development team (chipster@csc.fi).

How to cite Chipster?

Please cite the following paper when you use Chipster in your publications:

M Aleksi Kallio, Jarno T Tuimala, Taavi Hupponen, Petri Klemelä, Massimiliano Gentile, Ilari Scheinin, Mikko Koski, Janne Käki, Eija I Korpelainen: Chipster: user-friendly analysis software for microarray and other high-throughput data (2011) BMC Genomics 12: 507

Note that references for the third party analysis tools integrated in Chipster are available in the tool manual pages (accessed by the "More help" button in Chipster). Please also cite these when appropriate.
